About Me

I was a software build engineer at Microsoft and an important part of delivering quality builds to my customers. I worked on Windows, SQL Server, XBOX, and Surface products for 8 years. I built automation systems to streamline a software build process using Perl, Visual Basic, C#, CMD programming languages as well as build technologies CoreXT, Ant, Git, nmake, and MSBuild.

I have an undergraduate degree in Computer Science at Brigham Young University - Hawaii. I learned core computer science concepts and created a simplified Pascal compiler (grammar), designed an operating system (C and MIPS Assembly), implemented the TCP/IP protocol from scratch (bits and bytes manipulations), rendered 3D graphics (a lot of matrix computations).

I always wanted to learn more about web development because there are so many new technologies coming out every year and I need to stay sharp. I seem to do well and fully committed in an organized setting which was the main reason to get enrolled in the UW Full-stack coding bootcamp. On February 26, I successfully completed a full-stack coding bootcamp offered by University of Washington. It was a 3-month long full-time program concentrated on web development and technologies. I learned many web technologies (React, JQuery, Handlebars, DOM, Bootstrap, CSS, Node, Express, socket, Mongo NoSQL, IndexDB, Sequelize) and developed apps to get more practice.
